**주요 요소**:
- 헤더: 로고 및 주요 네비게이션 메뉴
- 성과 요약 카드 3개: 총 발송 수, 평균 클릭률, 평균 전환율
- **성과 데이터 연결 상태 카드**: Google Sheets 연동 상태 표시
  - 연결 상태 배지 (연결됨 / 미연결 / 오류)
  - 마지막 동기화 시간
  - "지금 동기화" 버튼
- 최근 캠페인 테이블: 제목, 발송일, 발송 시간, 발송 수, 주요 성과 지표
- CTA 버튼: "새 캠페인 만들기" (중앙 배치)

**연결 상태 배지 옵션**:
- **연결됨** (초록색 ●): Google Sheets가 정상적으로 연결됨
- **미연결** (회색 ●): 아직 Google Sheets를 연결하지 않음
- **오류** (빨간색 ●): 연결 오류 또는 권한 문제

- **타겟 고객**: 신규 회원, VIP 회원, 전체 회원, 해당 브랜드 구매 유관 행동 고객
- **톤앤매너**: 친근한, 모던한, 긴급한, 감사, 안내
